Mre_auto_screenshot

Basic app without interface to automaticaly create 10 screenshoot and save to storage card as bmp files for MRE platform mobile phone (including Nokia S30+). Application automaticali take screenshoot every 6 sec then exits. If need the program can be stopped by running the same program a second time. Tested on Nokia 225 with display resolution 240x320. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application files - "mre_auto_screenshot.vxp";
